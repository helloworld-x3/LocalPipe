# 实验设计：文化画像管线 vs 裸Prompt 对照实验

Hello World 团队 · 易点天下「AI创意本地化流水线」命题

## 1. 研究问题与假设

**研究问题**：搭载国别文化画像库的分层管线，产出的本地化创意是否显著优于直接向 LLM 下达本地化指令（裸Prompt）？

**假设 H1**：B组（管线组）在"地道感"评分上显著高于A组（裸Prompt组）
**假设 H2**：B组在"购买意愿"上不低于A组（重创作没有以牺牲营销效果为代价）
**假设 H0（零假设）**：两组无显著差异——若成立，画像库路线不成立，方案需调整

**学术先例**：MT Summit 2025（Unbabel）对 GPT-4o 的同类实验显示，20条广告转译中带文化上下文的 Prompt 在17条上优于裸Prompt。本实验为中文→多语种方向的复现与扩展。

## 2. 实验变量

- **自变量**：生成方式（A=裸Prompt直出 / B=四层管线：解构→画像重创作→质检→输出）
- **因变量**：地道感评分、购买意愿评分、盲测判别率（被认为"本地人写的"比例）
- **控制变量**：源文案相同、基座模型相同（两组用同一个LLM，排除模型差异）、目标语种相同、输出长度限制相同

## 3. 实验材料

- **源文案**：10条中文营销文案，覆盖5个品类（3C数码、美妆、食品饮料、服装、家居），每品类2条；一半含文化梗/情绪钩子（难样本），一半平铺直叙（易样本）
- **目标市场**：3个（建议：泰国、日本、美国——文化距离由近到远）
- **产出规模**：10条源文案 × 3市场 × 2组 = 60条待评样本

## 4. 评审设计

### 评审来源（三渠道）
| 渠道 | 负责人 | 目标人数 |
|------|--------|----------|
| 外国客户（真实消费者/从业者） | 唐启程 | 每市场2-3人 |
| Discord 区域/语言社区 | 孙文洁 | 每市场3-5人 |
| 语言交换App（HelloTalk/Tandem） | 乔唯一 | 每市场2-3人 |

每个市场合计 5-10 位母语评审。

### 盲测规则
1. 每位评审只看自己母语市场的样本
2. A/B 两组产出打乱顺序、去除任何组别标识
3. 每条样本只问三个问题（减轻评审负担，提高完成率）：
   - Q1 这段文案像本地人写的，还是翻译过来的？（本地人写的 / 翻译的 / 说不好）
   - Q2 自然程度打分 1-5（1=明显机翻，5=完全自然）
   - Q3 如果这是广告，你有兴趣继续看吗？（有 / 没有）
4. 每位评审最多评 20 条，防疲劳失真

### 数据记录
飞书多维表格统一收数：样本ID、组别（评后揭盲）、评审ID、渠道、Q1-Q3结果、评审备注。

## 5. 判定标准

- **B组胜出**：B组 Q2 均分高于A组 ≥ 0.5 分，且 Q1"本地人写的"比例高于A组 ≥ 15 个百分点 → 画像库路线成立
- **无显著差异**：两项指标差距均小于上述阈值 → 分析B组失效原因（画像质量？Prompt设计？），迭代后复测一轮；复测仍无差异则调整技术路线
- **分市场分析**：分别统计泰/日/美三个市场，观察文化距离与管线增益的关系（预期：文化距离越大，管线优势越明显）

## 7. 效度威胁与应对

- **评审样本偏差**：语言交换App用户偏年轻、客户偏商业视角 → 记录评审渠道，分渠道对比，结论注明适用范围
- **评审人数偏少**：每市场5-10人达不到统计显著性 → 定位为"定性验证+方向判断"，报告中如实说明，不夸大为严格统计结论
- **单模型局限**：结论只对所用基座模型成立 → 报告注明模型型号，留换模型复测接口
- **难易样本混淆**：平铺文案两组可能都好 → 分难/易样本统计，重点看难样本（含文化梗）上的差距
- **画像冷启动偏差**：日/美画像 v0.1 为 LLM 生成，可能携带模型训练数据中的文化刻板印象（如"日本人都很委婉""美国人都很直接"）。若偏差写进画像 → 重创作层放大 → 闭环反而加重偏差 → **盲测第一项任务即为母语者逐条审查画像内容，标记与实际文化不符的条目；校准前不用于批量生产**

2. 60条样本库 + 母语者评分数据——答辩时的核心证据
